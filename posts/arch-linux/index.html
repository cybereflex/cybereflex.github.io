Install Arch Linux

Hyper-V Desktop

通过 archinstall 安装系统

  1. 设置 Mirrors 为 China
  2. Disk Configuration 设置分区
  3. Bootloader 默认 systemd-boot 也可以切换为 grub
  4. 设置Root password
  5. Profile 选择 Desktop KDE-plasma
  6. Network Configuration 选择 use NetworkManager
  7. Additional Packages 可选择安装 noto-fonts fcitx5
  8. Timezone 设置为 Shanghai

高分辨率显示器 Hyper-V 不支持修改对应分辨率问题解决方案

首先关闭 Hyper-V 虚拟机

Set-VMVideo -VMName ArchLinux -HorizontalResolution 2560 -VerticalResolution 1440 -ResolutionType single

Set-VM -Name ArchLinux --EnhancedSessionTransportType HvSocket

wayland

  1. grub
sudo vim /etc/default/grub ## 在 options 后面追加 video=Virtual-1:2560x1440, 具体配置得看当前虚拟显示器名称和分辨率

sudo update-grub
  1. systemd-boot
sudo vim /boot/loader/entries/arch.conf ## options 最后追加 video=Virtual-1:2560x1440, 具体配置得看当前虚拟显示器名称和分辨率

修改完成后重启即可

x11

临时生效，重启后失效，可通过 xrandr 和 cvt 创建分辨率并设置

cvt 2560 1440

xrandr --newmode "2560x1440_60.00" ## 后面追加上面cvt 生成的 modeline

xrandr --addmode Virtual-1 "2560x1440_60.00" ## 虚拟显示器追加分辨率

xrandr --output Virtual-1 --mode "2560x1440_60.00" ## 应用新增的分辨率

永久生效

# 新增以下配置到 ~/.xprofile 或 ～/.xinitrc

xrandr --newmode "2560x1440_60.00" ## 后面追加上面对应分辨率 cvt 生成的 modeline

xrandr --addmode Virtual-1 "2560x1440_60.00"

xrandr --output Virtual-1 --mode "2560x1440_60.00"

Windows 双系统

预先从 Windows C 盘缩容 256G 的大小

连接无线网络

iwctl

device list 

station wlan0 scan

station wlan0 get-networks

station wlan0 connect $SSID

exit

同步时钟

hwclock --systohc --utc

timedatectl set-ntp true

磁盘分区与挂载

fdisk -l ## 列出磁盘情况，找到缩容的磁盘

cfdisk /dev/nvme0n1 ## 对 /dev/nvme0n1 进行分区，Swap 分区 4G，剩下容量为 Linux filesystem

mkswap /dev/nvme0n1p4 ## 格式化交换分区

mkfs.ext4 /dev/nvme0n1p5 ## 将 Linux filesystem 分区格式化为 ext4

mkdir /mnt/boot ## 创建 boot 目录准备挂载引导分区

mount /dev/nvme0n1p5 /mnt ## 挂载 Linux filesystem 到 /mnt 上

mount /dev/nvme0n1p1 /mnt/boot ## 将 Windows 的引导分区挂载到 /mnt/boot 目录上

swapon /dev/nvme0n1p4 ## 启用交换分区

系统与软件安装与配置

切换 pacman 源

reflector --country 'China' --sort rate --save /etc/pacman.d/mirrorlist

安装系统和一些工具

pacstrap /mnt base base-devel linux linux-firmware vim e2fsprogs ntfs-3g wget 


genfstab -U -p /mnt >> /mnt/etc/fstab ## 上一步安装完成后写入 fstab 文件
设置登陆用户
arch-chroot /mnt /bin/bash ## 进入安装好的系统

passwd  ## 设置 root 用户名密码

useradd -m -G wheel -s /bin/bash $USERNAME ## 新增普通用户

passwd $USERNAME ## 设置普通用户密码

visudo /etc/sudoers ## 编辑 sudoers 使 wheel 用户组下用户可使用 sudo 提权， 

# /etc/sudoers 取消下面内容的注释
%wheel ALL=(ALL) ALL
设置主机名和 hosts
echo archlinux >> /etc/hostname ## 设置主机名

vim /etc/hosts ## 写入 localhost 映射

# /etc/hosts 写入以下内容
127.0.0.1	localhost
::1		    localhost
127.0.1.1   archlinux.localdomain   archlinux
设置语言环境和时区
vim /etc/locale.gen

# /etc/locale.gen 取消下面内容的注释
en_US.UTF-8 UTF-8
zh_CN.UTF-8 UTF-8
zh_TW.UTF-8 UTF-8
ja_JP.UTF-8 UTF-8

locale-gen 

echo LANG=en_US.UTF-8 > /etc/locale.conf ## 设置当前语言环境为 en utf8

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime ## 设置时区为上海
安装微码和配置引导
pacman -S amd-ucode ## AMD cpu

pacman -S intel-ucode ## intel cpu

引导方式以下两种可二选一

[grub]

pacman -S grub efibootmgr os-prober

grub-mkconfig -o /boot/grub/grub.cfg

[systemd-boot]

bootctl install

vim /boot/loader/loader.conf ## 编辑此文件写入配置

# /boot/loader/loader.conf 配置如下
default arch.conf
timeout 3
console-mode keep
editor no

lsblk -f ## 获取根路径挂载磁盘的 UUID, 需要写下面配置文件的 options

vim /boot/loader/entries/arch.conf ## 创建此文件并写入以下配置

# /boot/loader/entries/arch.conf 配置如下
title   Arch Linux
linux   /vmlinuz-linux
initrd  /amd-ucode.img
initrd  /initramfs-linux.img
options root=UUID=fce2bf25-8543-495b-9104-c2f22749c9a7 rw
安装桌面环境
pacman -S plasma sddm konsole dolphin ark kate firefox networkmanager

systemctl enable sddm

systemctl enable NetworkManager

至此安装已完成，然后重启电脑，默认会引导进入 linux, 开机后可安装中文字体和中文输入法

安装 noto-fonts 和 fcitx5

pacman -S noto-fonts noto-fonts-emoji noto-fonts-cjk noto-fonts-extra

pacman -S fcitx5 fcitx5-im fcitx5-chinese-addons
